#here Motor Control System (TIA Simulator)

## Description
This project focuses on designing and implementing an industrial motor
control system using Siemens PLC in TIA Simulator.

## Key Features
- Motor start and stop control
- Manual and Automatic operation modes
- Emergency stop logic
- Overload and fault protection

## Tools
- Siemens PLC
- TIA Portal
- Ladder Logic (LAD)

## Project Type
Real PLC Project (TIA Simulator)
